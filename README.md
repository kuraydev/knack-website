# Knack website

Landing page and privacy policy for [Knack](https://knack.tools) — a native macOS toolkit.

- `index.html` — landing page
- `privacy.html` — privacy policy (linked from the Mac App Store listing)
- `styles.css` — shared stylesheet (mirrors the app's SwiftUI theme)
- `icon.png` — app icon

## Deploy

Hosted on GitHub Pages. Any push to `main` deploys automatically.

## Custom domain

Configured via `CNAME` (added after domain purchase).
